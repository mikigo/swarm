### 确认的关键决策

| 决策点 | 选择 |
|--------|------|
| 通信协议 | WebSocket + HTTP |
| 分发策略 | 动态按需，按文件粒度 |
| 代码同步 | 任务参数指定仓库，支持分支/Tag |
| 虚拟环境 | 每个任务独立创建 |
| 客户端参数 | 前缀区分，客户端转换 |
| 任务状态 | 只关心是否执行完，不关心用例通过率 |
| 心跳超时 | 任务重新分配，可配置 |
| CLI | 子命令 swarm server/client/run |
| CLI 任务列表 | rich 表格，可配置列、列宽、颜色 |
| 任务存储 | 每个任务单独 JSON 文件 |
